Объявляем переменную elementsCount, которая будет считать количество элементов в строке
Задаем нужный нам лимит по длине строк нового массива lengthLimit = 3
Запускаем цикл for, который перебирает элементы строки массива, пока не дойдет до третьего, где i = lengthLimit
С каждой итерацией i переменная elementsCount также итерируется, пока не доходит до значения lengthLimit
Теперь объявляем новый массив shortenedArray, в который в качетсве аргумента попадет переменная еlementsCount
Объявляем j=0 (индекс для запуска следующего цикла for)
В ходе цикла мы переприсваиваем значение элементов исходного массива в сокращенный, пока не доходим до элемента с индексом равным lengthLimit
Метод возвращает массив shortenedArray.
